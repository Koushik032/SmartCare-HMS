Receptionist Appointment Guide

This page helps receptionist users review appointment details, payment information, patient contact, booking status, and assign approximate appointment time when needed.

Workflow
  • Use the search box to find appointments by patient name, doctor name, contact, or hospital.
  • Use date filters to narrow down appointment records for a specific period.
  • If an appointment is waiting for receptionist action, assign an approximate appointment time.
  • Review payment method, status, bKash number, and transaction ID before confirming workflow.
